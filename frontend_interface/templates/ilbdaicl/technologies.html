Incorporated Technologies

GitHub
We used GitHub as our main code repository for version control.
Pivotal Tracker
Pivotal Tracker served as our project management tool in an Agile environment.
Cassandra
Cassandra stored the tweet data after filtering.
Storm
Storm handled the queuing of our Twitter API data stream and also handled the processing of the data stream.
Maven
Maven compiled our Cassandra database and our Storm package.
Amazon Web Services
AWS hosted the servers for all of our components.
Twitter
Using Twitter's API, we pulled tweets to filter for hateful sentiment across the country.
JSON
JSON lets us transfer data easily in a standard data structure.
Python
We used Python everywhere. Our sentiment analysis and its Storm bolt is in Python, and we used Python to talk to the database. Our front-end web-server also uses Python.
TextBlob
TextBlob, a Python library, allows us to do sentiment analysis on the content of tweets.
Django
Django Unchained. 'Nuff said.
C3 JS
C3JS rendered the visualization graphs of our tweet data.
Elm
We used Elm in conjunction with HTMl, JavaScript, and CSS to create the webpage.
HTML
We used HTML in conjunction with Elm, JavaScript, and CSS to create the webpage.
JavaScript
We used JavaScript for the interactive elements for the webpage.
GNIP
We moved to using GNIP to take advantage of their aggregates.
